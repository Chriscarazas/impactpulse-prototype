# Content Style Guide

## Voice

ImpactPulse is calm, specific, and accountable. It helps users make better judgments without pretending uncertainty has disappeared.

Use:

- "Estimated social value"
- "Approved evidence"
- "Needs review"
- "Methodology version"
- "Plausible range"
- "Most influential assumption"
- "Recommended next decision"

Avoid:

- "Impact unlocked"
- "AI-powered magic"
- "Guaranteed outcomes"
- "SDG aligned" without target and evidence
- "Benchmark leader" without comparability safeguards
- "Assured" unless external assurance has occurred

## Labels

Labels should name the user task or evidence state:

- Good: "Review transport assumption"
- Weak: "Details"
- Good: "Show evidence lineage"
- Weak: "Learn more"
- Good: "Reject SDG claim"
- Weak: "Decline"

## Methodology Help

Methodology help should answer:

- Why this matters
- What good evidence looks like
- What can go wrong
- Which result or report section it affects
- What action is recommended

## Claim Language

Use controlled claim levels:

1. Relevant to
2. Aligned with
3. Contributes to
4. Measured contribution to
5. Verified contribution to

Do not publish stronger language than the approved evidence supports.
